🚀 QUICK START - 3 SIMPLE STEPS
═══════════════════════════════════════════════════════════════════════════════

STEP 1: CREATE GITHUB REPOSITORY
─────────────────────────────────────────────────────────────────────────────
1. Go to https://github.com/moshe-stein24
2. Click "+" → "New repository"
3. Name: bucharest-job-search (or any name)
4. Visibility: ✅ PUBLIC (important!)
5. DON'T initialize with README
6. Click "Create repository"


STEP 2: PUSH YOUR CODE TO GITHUB
─────────────────────────────────────────────────────────────────────────────
Run these commands in terminal:

cd /mnt/user-data/outputs/job_search_website

# Add your GitHub repository
git remote add origin https://github.com/moshe-stein24/bucharest-job-search.git

# Push to GitHub
git push -u origin main

You'll need a Personal Access Token:
→ Go to: https://github.com/settings/tokens
→ Generate new token (classic)
→ Select "repo" scope
→ Copy token and use as password when pushing


STEP 3: ENABLE GITHUB PAGES (FREE HOSTING!)
─────────────────────────────────────────────────────────────────────────────
1. Go to your repository: https://github.com/moshe-stein24/bucharest-job-search
2. Click "Settings" → "Pages"
3. Source: Branch "main", folder "/ (root)"
4. Click "Save"
5. Wait 1-2 minutes

Your website will be live at:
🌐 https://moshe-stein24.github.io/bucharest-job-search/

🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: Can't push to GitHub
→ Use Personal Access Token, not password
→ Get token from: https://github.com/settings/tokens

Problem: GitHub Pages not showing
→ Wait 2-3 minutes after enabling
→ Check repository is PUBLIC
→ Verify branch is set to "main"

Problem: Map not loading
→ Check internet connection
→ Open browser console (F12)
→ Look for errors

Problem: Contact form not working
→ Ensure default email client is set
→ Try different browser
→ Consider FormSpree for production
